# Sistema de Gestión Software FJ - Fase 4

Este proyecto implementa un sistema de gestión de servicios (Salas, Equipos y Asesorías) aplicando los pilares de la Programación Orientada a Objetos (POO) en Python.

## 🛠️ Tecnologías Utilizadas
* Python 3.x
* Programación Orientada a Objetos (Herencia, Polimorfismo, Encapsulamiento, Abstracción)
* Logging para trazabilidad.

## 🚀 Cómo ejecutar
1. Clonar el repositorio.
2. Navegar a la carpeta `src`.
3. Ejecutar `python main.py`.
4. Los resultados se verán en consola y se guardarán en el archivo `logs.txt`.
